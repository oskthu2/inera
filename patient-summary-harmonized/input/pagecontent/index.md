### Inera EHDS Patient Summary Harmonized

Denna implementation guide sammanställer profilkrav för Patient Summary genom att harmonisera:

1. EURIDICE EU Health Data API (resursorienterad åtkomst)
2. HL7 EU EPS-artifakter
3. Xt-EHR EHDS Patient Summary logisk modell

Fokus ligger på:
- enhetliga kardinaliteter,
- tydligare kodverksstrategi,
- och spårbarhet för varje regel.
